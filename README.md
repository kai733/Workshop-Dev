# Workshop SAE

**Workshop SAE** est un thème WordPress sur mesure développé pour l'association Emanciper. Designé par des étudiants en 2ème année de MMI.

## Fonctionnalités Principales


*   **Gestion des Événements** : Un "Custom Post Type" (CPT) dédié pour gérer facilement l'agenda de l'association.
*   **Pages Modèles** : Des templates spécifiques pour les pages clés (Agenda, Contact, Adhésion, etc.).
*   **Navigation** : Menu principal personnalisable et fil d'ariane (breadcrumb) intégré pour une meilleure expérience utilisateur.
*   **Formatage de Date Intelligent** : Une fonction personnalisée pour afficher les dates des événements de manière esthétique (ex: JAN. 01).
